# HTML 从零到专家 - 完整学习路径

欢迎来到HTML完整学习体系！本课程将带你从零基础到专家水平。

## 📚 学习路径

### 第一阶段：基础篇（01-基础）
- **00-HTML实体与特殊字符.html** - HTML实体代码、特殊字符表示
- **01-文档结构与基本语法.html** - HTML文档的骨架和基本语法规则
- **02-文本元素与格式化.html** - 标题、段落、文本格式化标签
- **03-链接与导航.html** - 超链接、锚点、导航结构
- **04-图像与媒体基础.html** - img标签、图像优化
- **05-列表.html** - 有序列表、无序列表、定义列表
- **06-HTML全局属性.html** - id、class、data-*、contenteditable等全局属性

### 第二阶段：进阶篇（02-进阶）
- **06-表格.html** - table、tr、td、thead、tbody等表格元素
- **07-表单基础.html** - form、input、label等表单基础
- **08-表单高级.html** - 表单验证、高级输入类型、文件上传
- **09-多媒体.html** - audio、video标签
- **10-嵌入内容.html** - iframe、object、embed

### 第三阶段：高级篇（03-高级）
- **11-HTML5语义化.html** - header、nav、section、article等语义标签
- **12-元数据与SEO.html** - meta标签、SEO优化
- **13-Web组件基础.html** - 自定义元素、Shadow DOM基础
- **14-图形与动画.html** - Canvas和SVG基础
- **15-无障碍访问.html** - ARIA、无障碍最佳实践

### 第四阶段：实战篇（04-实战）
- **16-完整项目示例.html** - 综合性项目实战

## 🎯 学习建议

1. **按顺序学习**：每个文件都建立在前面知识的基础上
2. **动手实践**：每个示例都要自己编写和修改
3. **理解概念**：不要死记硬背，理解每个标签的语义和用途
4. **查看源码**：用浏览器开发者工具查看实际渲染效果
5. **不断实验**：修改代码，观察效果变化
