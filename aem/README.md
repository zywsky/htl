# Adobe Experience Manager (AEM) 完整学习路径

## 📚 学习目标
从零基础到 AEM 专家水平，掌握 AEM 的核心概念、开发技能和最佳实践。

## 🎯 学习路径概览

### 第一部分：AEM 基础概念和架构
- AEM 是什么
- JCR (Java Content Repository) 基础
- Apache Sling 框架
- AEM 架构概览
- 内容结构 (Content Structure)

### 第二部分：AEM 组件开发基础
- 组件结构
- 对话框 (Dialogs) 开发
- 组件模板
- 组件继承

### 第三部分：HTL (Sightly) 模板语言
- HTL 语法基础
- 数据绑定和上下文
- 表达式语言
- 模板和片段

### 第四部分：Sling 模型 (Sling Models)
- Sling Models 注解
- 资源适配
- 依赖注入
- 模型导出

### 第五部分：OSGi Bundles 和服务
- OSGi 基础
- OSGi 服务
- 事件监听器
- 调度任务

### 第六部分：AEM 工作流和流程
- 工作流模型
- 工作流步骤
- 流程步骤
- 工作流启动器

### 第七部分：AEM 内容服务和 API
- Content Services API
- REST API
- GraphQL API
- 内容片段

### 第八部分：高级主题和实践
- AEM Forms
- AEM Sites
- 多站点管理 (MSM)
- 性能优化
- 测试策略

## 🚀 快速开始

1. 按顺序阅读每个部分的文档
2. 运行和理解代码示例
3. 完成每部分的实践练习
4. 逐步构建完整的 AEM 应用

## 💡 学习建议

- **循序渐进**：按照顺序学习，每个部分都建立在前一部分的基础上
- **实践为主**：多写代码，多实验
- **理解原理**：不要只是复制代码，要理解为什么要这样写
- **查阅文档**：AEM 官方文档是最好的参考资源
