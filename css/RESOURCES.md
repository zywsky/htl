# 学习资源推荐

## 📚 官方文档

### MDN Web Docs
**网址：** https://developer.mozilla.org/zh-CN/docs/Web/CSS

**推荐理由：**
- 最权威的CSS文档
- 详细的语法说明和示例
- 浏览器兼容性信息
- 中文支持

**适合：** 查询语法、查找API、了解兼容性

---

## 🎓 在线教程

### CSS-Tricks
**网址：** https://css-tricks.com/

**推荐理由：**
- 高质量的CSS教程和文章
- 实用的代码片段
- 最新的CSS技术
- 英文网站，但内容非常优秀

**适合：** 深入学习、查找解决方案

### W3Schools
**网址：** https://www.w3schools.com/css/

**推荐理由：**
- 简单易懂的教程
- 在线编辑器可以立即测试
- 中文支持

**适合：** 初学者、快速查找语法

### freeCodeCamp
**网址：** https://www.freecodecamp.org/

**推荐理由：**
- 免费的完整课程
- 实践项目
- 社区支持

**适合：** 系统学习、项目实践

**使用场景：** 决定是否使用某个CSS特性，查看兼容性

**使用场景：** 快速测试想法、学习他人代码、创建演示

## 💡 学习建议

1. **从官方文档开始**：MDN是最权威的资源
2. **多动手实践**：理论结合实践才能掌握
3. **参考优秀作品**：学习别人的代码和设计
4. **加入社区**：与其他开发者交流学习
5. **持续更新**：CSS在不断进化，保持学习

---

**记住：最好的学习方式是实践！** 🚀
