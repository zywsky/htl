CSS盒模型详解

理解元素如何占据空间

1. 盒模型可视化

Content（内容）
Margin（外边距）- 透明
Border（边框）
Padding（内边距）
Content（内容）

2. content-box vs border-box

content-box（标准盒模型）

width: 200px

padding: 20px

border: 5px

实际宽度 = 200 + 20×2 + 5×2 = 250px

border-box（推荐）

width: 200px

padding: 20px

border: 5px

实际宽度 = 200px（内容区自动缩小）

3. Margin（外边距）

Box 1
margin-bottom: 20px
Box 2
margin-top: 30px
（合并后间距为30px）
水平margin不会合并
两个盒子之间的间距是 20px + 20px = 40px

4. Padding（内边距）

padding: 10px（四个方向）
padding: 10px 30px（上下 左右）
padding: 10px 20px 30px 40px（上 右 下 左）

5. Border（边框）

solid（实线）
dashed（虚线）
dotted（点线）
double（双线）
四个方向不同的边框

6. Border-radius（圆角）

border-radius: 10px
border-radius: 20px 10px
border-radius: 50%
（圆形）
border-radius: 10px / 20px
（椭圆）
