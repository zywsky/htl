# CSS 从零到专家 - 完整学习路径

## 📚 课程结构

本课程将带你从零基础到CSS专家水平，包含完整的代码示例和详细解释。

### 学习路径

1. **01-html-basics** - HTML基础：理解HTML结构和语义化标签
2. **02-css-basics** - CSS基础：语法、选择器、属性和值
3. **03-box-model** - 盒模型：理解元素如何占据空间
4. **04-layout-flexbox** - Flexbox布局：现代一维布局方案
5. **05-layout-grid** - Grid布局：强大的二维布局系统
6. **06-responsive-design** - 响应式设计：适配各种设备
7. **07-animations** - 动画和过渡：让页面动起来
8. **08-advanced-features** - 高级特性：CSS变量、函数、伪元素等
9. **09-practice-project** - 综合实战项目：创建完整的作品集网站

### 如何使用

每个目录都包含：
- `index.html` - 可运行的示例代码
- `styles.css` - 对应的样式文件
- `README.md` - 详细的概念解释

直接在浏览器中打开 `index.html` 查看效果，并阅读代码中的注释学习。

1. **循序渐进**：按顺序学习，每个章节都建立在前面的基础上
2. **动手实践**：修改代码，观察效果变化
3. **理解原理**：不仅要会用，更要理解为什么
4. **多做练习**：每学完一章，尝试创建自己的示例
