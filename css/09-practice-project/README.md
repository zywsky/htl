# 第九章：综合实战项目

## 项目目标

通过创建一个完整的响应式网站，整合前面学到的所有CSS知识。

## 项目要求

创建一个**个人作品集网站**，包含以下功能：

### 必须实现的功能

1. **响应式布局**
   - 使用Grid或Flexbox创建布局
   - 适配手机、平板、电脑
   - 使用媒体查询

2. **导航栏**
   - 固定顶部导航
   - 移动端汉堡菜单
   - 平滑滚动

3. **首页Hero区域**
   - 全屏背景
   - 居中的标题和按钮
   - 动画效果

4. **作品展示区域**
   - 卡片网格布局
   - 悬停效果
   - 图片响应式

5. **关于我区域**
   - 两列布局（桌面）
   - 单列布局（移动端）

6. **联系表单**
   - 表单样式
   - 输入框焦点效果
   - 按钮动画

7. **页脚**
   - 社交媒体链接
   - 版权信息

### 技术要求

- ✅ 使用CSS变量管理颜色和尺寸
- ✅ 使用Flexbox或Grid布局
- ✅ 添加过渡和动画效果
- ✅ 响应式设计
- ✅ 语义化HTML
- ✅ 代码注释清晰

## 实现步骤

### 步骤1：HTML结构

创建语义化的HTML结构，使用合适的标签。

### 步骤2：基础样式

- 重置样式
- 设置CSS变量
- 基础排版

### 步骤3：导航栏

- 固定定位
- Flexbox布局
- 移动端菜单

### 步骤4：Hero区域

- 全屏高度
- 居中内容
- 背景图片或渐变

### 步骤5：作品展示

- Grid布局
- 卡片设计
- 悬停效果

### 步骤6：响应式

- 媒体查询
- 移动端适配
- 测试不同设备

### 步骤7：动画和交互

- 过渡效果
- 滚动动画
- 按钮交互

## 评分标准

- **布局** (25分)：正确使用Flexbox/Grid，响应式设计
- **样式** (25分)：美观、一致的设计风格
- **交互** (20分)：动画、过渡、悬停效果
- **代码质量** (20分)：注释、结构、可维护性
- **功能完整性** (10分)：所有要求的功能都实现

## 扩展功能（加分项）

- 暗色主题切换
- 平滑滚动
- 加载动画
- 图片懒加载效果
- 自定义滚动条
- 更多动画效果

## 提示

1. **从简单开始**：先实现基本布局，再添加样式和动画
2. **移动优先**：先设计移动端，再扩展到桌面端
3. **测试**：在不同设备和浏览器上测试
4. **代码组织**：使用注释分隔不同部分
5. **参考**：可以参考优秀网站的设计，但要有自己的创意
