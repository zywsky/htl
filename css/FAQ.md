# 常见问题解答（FAQ）

## 基础问题

### Q1: CSS的三种引入方式有什么区别？

**A:** 
- **内联样式**：优先级最高，但难以维护，不能复用
- **内部样式表**：只能用于当前页面，适合单页应用
- **外部样式表**：可复用、易维护、可缓存，**推荐使用**

### Q2: 类选择器和ID选择器有什么区别？

**A:**
- **类选择器**（`.`）：可以多个元素共享，一个元素可以有多个类
- **ID选择器**（`#`）：应该是唯一的，一个页面中同一个ID只能出现一次
- **选择器优先级**：ID选择器（100分）> 类选择器（10分）

### Q3: margin和padding的区别是什么？

**A:**
- **margin**：元素**外部**的空间，透明，用于元素之间的间距
- **padding**：元素**内部**的空间，显示背景色，用于内容与边框的间距

### Q4: 什么是盒模型？

**A:** 盒模型描述了元素如何占据空间，由四部分组成（从内到外）：
1. Content（内容）
2. Padding（内边距）
3. Border（边框）
4. Margin（外边距）

## 布局问题

### Q5: Flexbox和Grid应该用哪个？

**A:**
- **Flexbox**：适合一维布局（行或列），如导航栏、按钮组
- **Grid**：适合二维布局（行和列），如页面整体布局、卡片网格
- **可以结合使用**：Grid用于整体布局，Flexbox用于组件内部

### Q6: 如何让元素水平居中？

**A:**
```css
/* 块级元素 */
.element {
    margin: 0 auto;
}

/* Flexbox */
.container {
    display: flex;
    justify-content: center;
}

/* Grid */
.container {
    display: grid;
    justify-items: center;
}
```

### Q7: 如何让元素完全居中（水平和垂直）？

**A:**
```css
/* Flexbox（推荐） */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

/* Grid */
.container {
    display: grid;
    place-items: center;
}

/* 绝对定位 */
.element {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
```

### Q8: 为什么我的Grid布局不换行？

**A:** 检查以下几点：
1. 是否设置了 `grid-template-columns` 的固定宽度
2. 使用 `repeat(auto-fit, minmax(250px, 1fr))` 实现自动换行
3. 检查容器的宽度是否足够

## 响应式问题

### Q9: 如何让图片响应式？

**A:**
```css
img {
    max-width: 100%;
    height: auto; /* 保持宽高比 */
}
```

### Q10: 媒体查询的断点应该怎么设置？

**A:** 常见的断点：
- 手机：`max-width: 575px`
- 大手机：`576px - 767px`
- 平板：`768px - 991px`
- 小电脑：`992px - 1199px`
- 大电脑：`min-width: 1200px`

**建议**：根据你的内容需要设置断点，不要盲目跟随。

### Q11: 移动优先是什么意思？

**A:** 移动优先是指：
1. 先为移动设备编写样式（默认样式）
2. 使用 `min-width` 媒体查询逐步增强
3. 这样代码更简洁，性能更好

## 动画问题

### Q12: transition和animation的区别？

**A:**
- **transition**：简单的属性变化过渡，需要触发（如hover）
- **animation**：复杂的多阶段动画，可以自动播放、循环

### Q13: 如何让动画更流畅？

**A:**
1. 使用 `transform` 和 `opacity`（不会触发重排）
2. 避免使用 `left`、`top`、`width`、`height`（会触发重排）
3. 使用 `will-change` 提示浏览器
4. 减少动画的复杂度

### Q14: 如何暂停动画？

**A:**
```css
.element {
    animation-play-state: paused;
}
```

## 选择器问题

### Q15: 选择器优先级如何计算？

**A:** 优先级分数：
- 内联样式：1000分
- ID选择器：100分
- 类选择器、属性选择器、伪类：10分
- 元素选择器、伪元素：1分

分数越高，优先级越高。如果分数相同，后定义的规则覆盖前面的。

### Q16: :nth-child() 如何使用？

**A:**
```css
li:nth-child(3) { }        /* 第3个 */
li:nth-child(3n) { }       /* 每3个 */
li:nth-child(odd) { }      /* 奇数 */
li:nth-child(even) { }     /* 偶数 */
li:nth-child(-n+3) { }     /* 前3个 */
```

## 性能问题

### Q17: CSS会影响性能吗？

**A:** 会，但影响较小。优化建议：
1. 避免过度嵌套的选择器
2. 使用 `transform` 代替位置属性
3. 减少重排和重绘
4. 使用CSS变量减少重复代码

### Q18: 如何减少CSS文件大小？

**A:**
1. 删除未使用的样式
2. 压缩CSS文件
3. 使用CSS变量减少重复
4. 合并相同的样式规则

## 兼容性问题

### Q19: 如何处理浏览器兼容性？

**A:**
1. 使用 [Can I Use](https://caniuse.com/) 检查兼容性
2. 使用 Autoprefixer 自动添加前缀
3. 提供降级方案
4. 使用 Modernizr 检测特性支持

### Q20: CSS Grid在所有浏览器都支持吗？

**A:** 现代浏览器都支持。如果需要支持旧浏览器：
1. 使用 `@supports` 提供降级方案
2. 使用 Flexbox 作为后备方案

## 调试问题

### Q21: 样式不生效怎么办？

**A:** 检查清单：
1. 检查选择器是否正确
2. 检查优先级是否被覆盖
3. 检查是否有拼写错误
4. 使用浏览器开发者工具检查
5. 检查是否有语法错误

### Q22: 如何使用浏览器开发者工具？

**A:**
1. 按 F12 打开开发者工具
2. 点击元素检查器（左上角图标）
3. 选择要检查的元素
4. 在右侧样式面板查看和修改样式
5. 查看计算后的样式值

## 最佳实践

### Q23: CSS代码应该怎么组织？

**A:**
1. 使用注释分隔不同部分
2. 按功能分组（布局、组件、工具类等）
3. 使用CSS变量管理颜色和尺寸
4. 保持一致的命名规范

### Q24: 应该使用CSS框架吗？

**A:**
- **初学者**：先学好原生CSS，理解原理
- **有经验后**：可以使用框架（Bootstrap、Tailwind CSS）提高效率
- **建议**：理解框架的原理，不要盲目使用

### Q25: 如何学习CSS？

**A:**
1. **循序渐进**：从基础到高级
2. **动手实践**：多写代码，多修改
3. **理解原理**：不仅要会用，更要理解为什么
4. **参考优秀作品**：学习别人的代码
5. **持续学习**：CSS在不断更新，保持学习
